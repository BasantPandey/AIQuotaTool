# Service Logos

Drop the brand logo files here. The `ServiceHeader` component loads them automatically.

| File | Service | Recommended format |
|---|---|---|
| `claude.svg` | Anthropic Claude | SVG (coral/orange `#CC785C`) |
| `copilot.svg` | GitHub Copilot | SVG |
| `codex.svg` | OpenAI Codex | SVG |

PNG at 48×48 also accepted — just name them `claude.png`, `copilot.png`, `codex.png`.

These files are gitignored (`packages/ui/src/assets/logos/*.svg` and `*.png`).
